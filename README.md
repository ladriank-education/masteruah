> Como no tengo ningun nuevo cambio en el repositorio primero creare un fichero.
> Para ello, usare el siguiente comando:
>>`touch fichero`

>>`git add fichero`
![imagen](.img/2.png)

>>`git commit -m "Commit Inicial"`
